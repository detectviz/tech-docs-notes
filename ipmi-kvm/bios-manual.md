**11.2 BMC 網路組態**

此組態中的子項目可讓您組態 BMC 網路參數。
**組態 IPV4 支援**

**LAN 通道 1**

**組態位址來源**

此項目可讓您靜態或動態（透過 BIOS 或 BMC）組態 LAN 通道參數。**---[未指定---]** 選項在 BIOS 階段不會修改任何 BMC 網路參數。

組態選項：---[未指定---] ---[靜態---] ---[DynamicBmcDhcp---]

只有在 **組態位址來源** 設定為 **---[靜態---]** 時，才能使用下列項目。

**站台 IP 位址**

可讓您設定站台 IP 位址。

**子網路遮罩**

可讓您設定子網路遮罩。建議您使用與作業系統網路上所用網路卡指定的相同子網路遮罩。

**路由器 IP 位址**

可讓您設定路由器 IP 位址。

**路由器 MAC 位址**

可讓您設定路由器 MAC 位址。
