## 安全資訊

### 電氣安全

- 為防止觸電危險，在重新安置系統前，請從電源插座拔下電源線。
- 在向系統新增或移除裝置時，請確保在連接訊號線之前，已拔下裝置的電源線。如果可能，在新增裝置前，請先拔下現有系統的所有電源線。
- 在連接或移除主機板的訊號線之前，請確保所有電源線均已拔下。
- 在使用轉接器或延長線之前，請尋求專業協助。這些裝置可能會中斷接地電路。
- 請確保您的電源供應器設定為您所在地區的正確電壓。如果您不確定所用電源插座的電壓，請聯繫您當地的電力公司。
- 如果電源供應器損壞，請勿自行修理。請聯繫合格的維修技術人員或您的零售商。

### 操作安全

- 在安裝主機板並在其上新增裝置之前，請仔細閱讀包裝內的所有手冊。
- 使用產品前，請確保所有纜線均已正確連接，且電源線未損壞。如果發現任何損壞，請立即聯繫您的經銷商。
- 為避免短路，請將迴紋針、螺絲和釘書針遠離連接器、插槽、插座和電路。
- 避免灰塵、濕度和極端溫度。請勿將產品放置在任何可能變濕的區域。
- 將產品放置在穩定的表面上。
- 如果您遇到產品的技術問題，請聯繫合格的維修技術人員或您的零售商。
- 您的主機板只能在環境溫度介於 0°C 至 40°C 的環境中使用。

### 鈕扣/硬幣電池安全資訊

**警告**
請將本產品放在兒童無法觸及的地方。吞嚥可能導致化學灼傷、軟組織穿孔和死亡。嚴重灼傷可在攝入後 2 小時內發生。請立即就醫。

## 第 1 章：產品介紹

### 1.1 開始之前

在安裝主機板組件或變更任何主機板設定之前，請注意以下預防措施。

- 在觸摸任何組件之前，請從牆壁插座拔下電源線。
- 在處理組件之前，請使用接地腕帶或觸摸安全接地的物體或金屬物體（例如電源供應器外殼），以避免因靜電而損壞組件。
- 握住組件的邊緣，以避免觸摸其上的 IC。
- 每當您卸載任何組件時，請將其放置在接地的防靜電墊上或組件附帶的袋子中。
- 在安裝或移除任何組件之前，請確保 ATX 電源供應器已關閉或電源線已從電源供應器上拔下。否則可能會對主機板、周邊設備或組件造成嚴重損壞。

- CPU 僅能以一個正確的方向安裝。請勿強行將 CPU 插入插槽。
- 安裝 CPU 前，請確保所有電源線均已拔下。
- 購買主機板時，請確保 PnP 保護蓋位於插槽上，且插槽接點未彎曲。
- 安裝主機板後，請保留保護蓋。
- 產品保固不涵蓋因不正確的 CPU 安裝/移除，或 PnP 保護蓋的錯置/遺失/不正確移除而導致的插槽接點損壞。

## 第 2 章：基本安裝

### 2.1 組建您的個人電腦系統

#### 2.1.1 CPU 安裝
- 僅安裝專為 LGA1700 插槽設計的 CPU。
- 華碩不承擔因不正確的 CPU 安裝/移除造成的損壞。

#### 2.1.2 散熱系統安裝
- 必要時，在安裝散熱系統之前，請將散熱介面材料塗抹在 CPU 散熱系統和 CPU 上。
- 建議在 Intel® 600 系列主機板上使用與 LGA1700 相容的散熱系統。

#### 2.1.3 DIMM 安裝
- 按照主機板上的圖示安裝 DIMM。

#### 2.1.4 M.2 安裝
- 按照步驟鬆開散熱器螺絲、安裝 M.2，然後重新固定散熱器。

#### 2.1.5 主機板安裝
- 將主機板放入機箱，確保後置 I/O 連接埠與機箱的後置 I/O 面板對齊，然後用九顆螺絲固定。

#### 2.1.6 ATX 電源連接
- 連接 24-pin 主電源、8-pin 和/或 4-pin CPU 電源，以及 6-pin PCIe 電源連接器。

#### 2.1.7 SATA 裝置連接
- 將 SATA 裝置連接到 SATA 連接埠。

#### 2.1.8 SlimSAS 連接
- 使用 SlimSAS 連接埠連接 NVM Express 儲存裝置。

#### 2.1.9 前置 I/O 連接器
- 連接前面板、USB 3.2 Gen 1、USB 3.2 Gen 2x2 Type-C® 和 USB 2.0 連接器。

#### 2.1.10 擴充卡安裝
- 安裝 PCIe x16、PCIe x1 和 Thunderbolt™ 系列卡。

#### 2.1.11 M.2 Wi-Fi 模組與天線安裝
- 安裝 M.2 Wi-Fi 模組並連接天線。

### 2.2 BIOS 更新公用程式

**BIOS FlashBack™** 可讓您輕鬆更新 BIOS，無需進入現有的 BIOS 或作業系統。

**若要使用 BIOS FlashBack™：**
1. 從華碩支援網站下載最新的 BIOS 版本。
2. 根據您的主機板型號重新命名檔案 (`PWW680AI.CAP` 或 `PWW680A.CAP`)，或使用 `BIOSRenamer.exe`。
3. 將 24-pin 電源連接器插入主機板並開啟電源供應器。將 USB 儲存裝置插入 BIOS FlashBack™ 連接埠。
4. 按住 BIOS FlashBack™ 按鈕三秒，直到 LED 閃爍三次。
5. 等待指示燈熄滅，表示更新完成。

### 2.3 主機板後方與音訊連接

#### 2.3.1 後方 I/O 連接
- DisplayPort
- USB 2.0
- Intel® 2.5Gb 乙太網路
- HDMI®
- VGA
- USB 3.2 Gen 1 Type-A
- USB 3.2 Gen 2 Type-A
- USB 3.2 Gen 2 Type-C®
- BIOS FlashBack™ 按鈕

#### 2.3.2 音訊 I/O 連接
根據您的喇叭設定（2、4、5.1 或 7.1 聲道）連接音訊裝置。

### 2.4 首次啟動
1. 完成所有連接後，裝回系統機殼蓋。
2. 連接電源線並開啟顯示器和系統電源。
3. 系統會執行開機自我測試 (POST)。
4. 開機時，按 `<Delete>` 鍵以進入 BIOS 設定。

### 2.5 關閉電腦
按下電源按鈕可將系統置於睡眠模式或軟關機模式。

## 第 3 章：BIOS 與 RAID 支援

### 3.1 認識 BIOS
華碩 UEFI BIOS 提供友善的介面，可實現滑鼠輸入。BIOS 將系統硬體設定儲存在主機板 CMOS 中。不當的 BIOS 設定可能會導致不穩定或開機失敗。

### 3.2 BIOS 設定程式
開機時按 `<Delete>` 或 `<F2>` 進入 BIOS 設定。BIOS 設定程式有 **EZ 模式** 和 **進階模式**。

### 3.3 華碩 EZ Flash 3
此功能可讓您在 BIOS 中更新 BIOS。
1. 將包含最新 BIOS 檔案的 USB 隨身碟插入 USB 連接埠。
2. 進入 BIOS 進階模式，前往 **工具** > **華碩 EZ Flash 3 公用程式**。
3. 選擇 USB 隨身碟和 BIOS 檔案以開始更新。

### 3.4 華碩 CrashFree BIOS 3
此工具可在 BIOS 檔案損毀時還原 BIOS。
1. 下載最新 BIOS 並將其重新命名為 `asus.cap` 或 `PWW680AI.cap`，然後複製到 USB 隨身碟。
2. 開啟系統並插入 USB 隨身碟。
3. 公用程式會自動還原 BIOS。

### 3.5 RAID 組態
主機板支援 PCIe RAID 0/1/5/10 和 SATA RAID 0/1/5/10 組態。

- **RAID 0 (資料分段):** 提高資料存取和儲存速度。
- **RAID 1 (資料鏡像):** 提供資料保護。
- **RAID 5:** 結合效能、容錯能力和儲存容量。
- **RAID 10:** 結合 RAID 0 和 RAID 1 的優點。
