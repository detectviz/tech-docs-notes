# `argocd-dex` 指令參考

## argocd-dex

Argo CD 使用的 argocd-dex 工具

### 摘要

argocd-dex 具有 Argo CD 使用的內部公用程式工具

```
argocd-dex [flags]
```

### 選項

```
  -h, --help   argocd-dex 的說明
```
