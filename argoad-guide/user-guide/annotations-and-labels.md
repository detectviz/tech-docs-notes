# Argo CD 使用的註解和標籤

## 註解

| 註解索引鍵                             | 目標資源 | 可能的值                                                                                   | 說明                                                                                                                                                                                                  |
|--------------------------------------------|---------------------|---------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| argocd.argoproj.io/application-set-refresh | ApplicationSet      | `"true"`                                                                                          | 當 ApplicationSet 被 Webhook 要求重新整理時新增。ApplicationSet 控制器會在協調結束時移除此註解。                                              |
| argocd.argoproj.io/compare-options         | 任何                 | [請參閱比較選項文件](compare-options.md)                                                    | 設定應用程式的目前狀態如何與其期望狀態進行比較。                                                                                                                                      |
| argocd.argoproj.io/hook                    | 任何                 | [請參閱資源掛鉤文件](resource_hooks.md)                                                      | 用於設定[資源掛鉤](resource_hooks.md)。                                                                                                                                                       |
| argocd.argoproj.io/hook-delete-policy      | 任何                 | [請參閱資源掛鉤文件](resource_hooks.md#hook-deletion-policies)                               | 用於設定[資源掛鉤的刪除策略](resource_hooks.md#hook-deletion-policies)。                                                                                                                   |
| argocd.argoproj.io/manifest-generate-paths | 應用程式         | [請參閱擴充文件](../operator-manual/high_availability.md#webhook-and-manifest-paths-annotation) | 用於避免不必要的應用程式重新整理，尤其是在單一儲存庫中。                                                                                                                                   |
| argocd.argoproj.io/refresh                 | 應用程式         | `normal`, `hard`                                                                                  | 表示應用程式需要重新整理。在應用程式重新整理後由應用程式控制器移除。值 `"hard"` 表示在重新整理之前應使資訊清單快取和目標叢集狀態快取失效。 |
| argocd.argoproj.io/skip-reconcile          | 應用程式         | `"true"`                                                                                          | 指示 Argo CD 應用程式控制器不應協調該應用程式。請參閱[跳過協調文件](skip_reconcile.md)以了解使用案例。                                      |
| argocd.argoproj.io/sync-options            | 任何                 | [請參閱同步選項文件](sync-options.md)                                                          | 提供各種設定以決定應用程式的資源如何同步。                                                                                                                       |
| argocd.argoproj.io/sync-wave               | 任何                 | [請參閱同步波文件](sync-waves.md)                                                              |                                                                                                                                                                                                              |
| argocd.argoproj.io/tracking-id             | 任何                 | 任何                                                                                               | 由 Argo CD 用於追蹤其管理的資源。詳情請參閱[資源追蹤文件](resource_tracking.md)。                                                                                               |
| argocd.argoproj.io/ignore-resource-updates | 任何                 | `"true"`, `false`                                                                                  | 由 Argo CD 用於忽略資源更新。詳情請參閱[協調文件](..%2Foperator-manual%2Freconcile.md)reconcile_docs。                                                                             |
| link.argocd.argoproj.io/{some link name}   | 任何                 | 一個 http(s) URL                                                                                    | 在 Argo CD UI 中為資源新增一個連結。詳情請參閱[外部 URL 文件](external-url.md)。                                                                                                        |
| pref.argocd.argoproj.io/default-pod-sort   | 應用程式         | [請參閱 UI 自訂文件](../operator-manual/ui-customization.md)                               | 設定應用程式的預設群組機制。                                                                                                                                                           |
| pref.argocd.argoproj.io/default-view       | 應用程式         | [請參閱 UI 自訂文件](../operator-manual/ui-customization.md)                               | 設定應用程式的預設檢視模式 (例如「樹狀」或「清單」)                                                                                                                                             |

## 標籤

| 標籤索引鍵                      | 目標資源 | 可能的值                                      | 說明                                                                                                                                                                                                                                                                       |
|--------------------------------|---------------------|------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| argocd.argoproj.io/instance    | 應用程式         | 任何                                                  | 建議的追蹤標籤，以[避免與其他使用 `app.kubernetes.io/instance` 的工具發生衝突](../faq.md#why-is-my-app-out-of-sync-even-after-syncing)。                                                                                                                  |
| argocd.argoproj.io/secret-type | Secret              | `cluster`, `repository`, `repo-creds`, `scm-creds` | 識別 Argo CD 使用的某些類型的 Secret。有關前三種的詳細資訊，請參閱[宣告式設定文件](../operator-manual/declarative-setup.md)；有關最後一種的詳細資訊，請參閱[任何命名空間中的 AppSet 文件](../operator-manual/applicationset/Appset-Any-Namespace.md)。 |
