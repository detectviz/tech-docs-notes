# 翻譯插件

Grafana UI 可以[以多種語言顯示](https://grafana.com/docs/grafana/latest/administration/organization-preferences/#change-grafana-language)。您可以在 GitHub 上找到[可用語言列表](https://github.com/grafana/grafana/blob/main/packages/grafana-i18n/src/constants.ts)。

預設情況下，插件僅提供英文版本，當您變更語言設定時不會被翻譯。若要了解如何翻譯您的插件，請參閱[翻譯您的插件](../how-to-guides/plugin-internationalization.md)操作指南。

:::note
Grafana 使用 [i18next 國際化框架](https://www.i18next.com/)進行翻譯。
:::
