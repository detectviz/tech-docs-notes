# 資源

這裡有一些與插件相關的額外資源，可以讓您的插件開發之旅更加順利。

- [Grafana 開發者入口網站](https://grafana.com/developers/) - 插件文件、Scenes API、React 元件庫、Data Plane 合約等的首頁。
- [Grafana 設計系統](https://grafana.com/developers/saga/about/overview/) - Saga 是我們的設計系統，為所有 Grafana Labs 產品建立了共享的視覺語言。
- [Grafana API 參考](https://grafana.com/docs/grafana/latest/developers/http_api/) - Grafana 的後端 HTTP API。
- [Grafana 社群](https://community.grafana.com/) - 關於插件開發和其他主題的官方 Grafana Labs 論壇。
- [社群 Slack](https://slack.grafana.com/) - 在 Grafana Labs 社群 Slack 上討論插件開發。
- [Grafana Labs 部落格](https://grafana.com/blog/) - 最新的新聞、功能和操作指南。
- [Grafana 插件目錄](https://grafana.com/grafana/plugins/) - 尋找許多熱門插件的範例。
- [貢獻者指南](https://grafana.com/docs/grafana/latest/developers/contribute/) - 為 Grafana 做出貢獻的開發者指南。
