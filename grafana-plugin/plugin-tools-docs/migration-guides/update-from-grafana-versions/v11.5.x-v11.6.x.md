# 將插件從 Grafana 11.5.x 版遷移至 11.6.x 版

請遵循以下說明，將插件從 Grafana 11.5.x 版遷移至 11.6.x 版。

## @grafana/runtime 中的 usePluginComponents 選項已變更

由於意外使用了不正確的已棄用類型，我們更新了 hook 的選項 TypeScript 類型。此變更已透過 [PR 100949](https://github.com/grafana/grafana/pull/100949) 提交至核心 `grafana` 儲存庫。

**建議措施：**

- 呼叫 `usePluginComponents` 時移除 `context` 屬性。`context` 應作為 props 傳遞給每個元件。

**請注意：**
這是建置時錯誤，不會對 Grafana 的執行階段產生任何影響。
