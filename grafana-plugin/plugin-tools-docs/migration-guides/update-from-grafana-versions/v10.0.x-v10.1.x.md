# 將插件從 Grafana 10.0.x 版遷移至 10.1.x 版

請遵循以下說明，將插件從 Grafana 10.0.x 版遷移至 10.1.x 版。

## grafana-ui 中 IconButton 元件的無障礙更新

由於無障礙問題，我們更新了元件的 TypeScript 介面。此變更已透過 [PR 69699](https://github.com/grafana/grafana/pull/69699) 提交至核心 `grafana` 儲存庫。

如果您在插件中使用 IconButton 元件，您將會收到與此變更相關的 TypeScript 錯誤。

**建議措施：**

- 檢閱您插件中 IconButton 的使用案例。
- 新增一個有意義的工具提示，元件也會將其用作 aria-label。
- 另一個選項是設定 aria-label。在這種情況下，不會顯示工具提示。

**請注意：**
IconButton 先前有一個名為 `ariaLabel` 的屬性，此變更後已棄用。您現在可以改用一般的 `aria-label` 屬性。
