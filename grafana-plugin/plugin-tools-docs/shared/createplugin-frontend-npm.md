### `npx run dev`

以開發模式建置插件，並監看原始碼的變更以重新建置插件。

### `npx run build`

為生產環境建置插件。輸出可在 `./dist` 目錄中找到。

### `npx run test`

以監看模式啟動測試執行器。

### `npx run e2e`

啟動端對端測試執行器。在執行端對端測試之前，請確保您已執行一個 Grafana 執行個體。

### `npx run lint`

對原始碼執行 linter，以驗證您是否已遵守格式和樣式規則。此指令也會報告已棄用程式碼的使用警告。

### `npx run lint:fix`

以修復模式執行 linter，以自動格式化前端程式碼。
