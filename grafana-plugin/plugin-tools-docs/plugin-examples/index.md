範例插件可讓您深入了解[幾種類型](../key-concepts/plugin-types-usage)的 Grafana 插件的結構、功能和實作細節。這些是真實世界的程式碼範例，您可以在您的專案中研究甚至重複使用。

以下列表描述了一些我們受歡迎的插件。

## 面板插件

- [**panel-basic:**](https://github.com/grafana/grafana-plugin-examples/tree/main/examples/panel-basic) 這是建立一個簡單面板插件以視覺化資料、導覽儀表板或控制裝置的良好起點。

## 資料來源插件

- [**datasource-basic:**](https://github.com/grafana/grafana-plugin-examples/tree/main/examples/datasource-basic) 此應用程式展示如何將新的資料來源（包括內部指標應用程式）整合至資料來源插件中。

## 應用程式插件

- [**app-basic:**](https://github.com/grafana/grafana-plugin-examples/blob/main/examples/app-basic/) 此應用程式插件展示如何新增基本功能，例如導覽、路由、樣式和設定。
